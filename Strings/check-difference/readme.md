# find the difference

You are given two strings s and t.

String t is generated by random shuffling string s and then add one more letter at a random position.

Return the letter that was added to t.

 

Example 1:

Input: s = "abcd", t = "abcde"
Output: "e"
Explanation: 'e' is the letter that was added.
Example 2:

Input: s = "", t = "y"
Output: "y"


## Problem

Do strings diye hai s aur t
👉 t = s + ek extra character

Find karo wo extra character

##  Approach 1: Sum Method (ASCII Addition)
 ### Idea
Sab characters ka ASCII sum add karo t me

Fir s ke characters subtract karo

Jo bachega wahi extra char hai

```java
int total = 0;

for(int i = 0; i < t.length(); i++){
    total += t.charAt(i);
}

for(int i = 0; i < s.length(); i++){
    total -= s.charAt(i);
}

return (char) total;

```

✅ Pros
Simple logic
Easy to understand

❌ Cons
Thoda heavy (addition/subtraction)
Overflow ka risk (large input me)


## Approach 2: XOR Method (Best 🚀)
### Idea
Same characters XOR hone pe cancel ho jaate hain
Sirf extra character bach jaata hai

👉 Property:

a ^ a = 0
a ^ 0 = a

```java 
int xor = 0;

for(char c : s.toCharArray()){
    xor ^= c;
}

for(char c : t.toCharArray()){
    xor ^= c;
}

return (char) xor;
```

🔹 Example
s = "abc"
t = "abcc"

Extra character → 'c'

🔹 Key Points
XOR method fast + efficient hai
Space complexity → O(1)
Time complexity → O(n)
